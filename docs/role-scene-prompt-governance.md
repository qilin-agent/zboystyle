# BoyStyle 角色与场景 Prompt 规范

这份规范用于约束站内后续的人物设定、出图提示词、题图策划、角色化文案与相关内容开发，避免内容再次滑向空泛的“年轻男生穿搭”“帅气男大”“制服性感”标签堆砌。

BoyStyle 需要的人物内容，不是脱离环境的摆拍幻想，而是**身份明确、场景明确、动作明确、服装明确、身体状态明确**的年轻男性角色表达。

## 一、总原则

所有角色化 prompt、人物化内容与场景化图文策划，必须优先满足以下顺序：

1. **先有身份**
2. **再有场景**
3. **再有动作**
4. **再有服装结构**
5. **最后才是性感、氛围或风格包装**

不能反过来先写“性感”“男大感”“制服诱惑”，再临时给它补一个不成立的场景。

## 二、核心目标

BoyStyle 的角色与场景 prompt 应同时满足：
- 年轻男性气质成立
- 青春感成立
- 场景可视化、可拍摄、可写作
- 服装与身份之间有逻辑关系
- 动作能支撑人物状态
- 吸引力来自具体语境，而不是空洞标签

最终输出要让人一眼看懂：
- 他是谁
- 他在什么地方
- 他正在做什么
- 他为什么有吸引力
- 这种吸引力来自哪里

## 三、允许的核心人物方向

### 1. 校园青春线
优先角色：
- 男大学生
- 校园社团成员
- 图书馆自习男生
- 校队成员
- 实习中的大学生
- 宿舍 / 教学楼 / 操场中的校园男生

关键词：
- 干净
- 青春
- 少年感
- 校园通勤
- 真实生活流
- 轻运动气息
- 不刻意摆拍

### 2. 青春运动线
优先角色：
- 篮球少年
- 足球少年
- 跑步男生
- 游泳队男生
- 健身训练生
- 校队运动员
- 体育馆日常训练角色

关键词：
- 运动后状态
- 身体热度
- 拉伸、出汗、呼吸感
- 肩背线条
- 青春活力
- 训练中的克制性感

### 3. 年轻职业男性线
优先角色：
- 办公室职员
- 实习生
- 教师
- 快递员
- 餐饮服务员
- 安保
- 商业服务业中的年轻男性

关键词：
- 职业身份
- 通勤状态
- 制服或工装结构
- 服务关系感
- 劳动感
- 下班前后状态
- 秩序感与克制感

## 四、必须保留的气质方向

即使扩展到职业与制服内容，仍应保留 BoyStyle 原有的以下核心气质：
- 青春感
- 男大学生气质
- 运动感
- 轻生活流
- 干净、顺眼、真实
- 东亚青年男性语境
- 可模仿、可想象、可进入的现实场景

不允许内容整体滑向：
- 过度油腻
- 纯 fetish 符号化
- 脱离现实的职业 cosplay
- 没有生活逻辑的空壳制服幻想

## 五、场景设计要求

### 1. 场景必须具体
不要只写：
- 在工作
- 在校园
- 在运动
- 在办公室

必须写成可见空间，例如：
- 图书馆靠窗长桌
- 晚自习后教学楼走廊
- 操场边看台
- 篮球训练后的场边长椅
- 写字楼电梯口
- 办公室工位之间的走道
- 餐厅后场备餐区
- 校门口安保岗亭
- 快递配送点和楼下取件处
- 教师办公室与讲台之间

### 2. 场景必须支撑动作
一个好场景不是背景板，而是动作发生的理由。

例如：
- 图书馆 → 低头写字、趴桌短休、翻书、摘耳机
- 篮球场边 → 撩衣摆擦汗、仰头喝水、撑膝喘气
- 办公桌前 → 松领带、整理文件、揉肩、端咖啡
- 快递点 → 弯腰搬箱、扫码签收、抬手整理衣领
- 餐厅后场 → 俯身摆盘、擦手、收工后放松
- 教室讲台 → 写板书、翻教材、扶眼镜、停顿讲解

## 六、动作设计要求

动作必须带有人物性，不能只是“站着很帅”“坐着很性感”。

### 1. 校园男大学生动作
优先动作：
- 背包快步赶课
- 单手抱书
- 低头写字
- 靠栏杆喝饮料
- 趴在桌上短暂休息
- 打完球后坐在台阶上喘气
- 弯腰系鞋带
- 撩起衣摆擦汗
- 抬手整理头发
- 课后卷袖子搬东西

### 2. 青春运动角色动作
优先动作：
- 热身跑
- 拉伸腿部和肩背
- 双手撑膝喘气
- 坐在场边仰头喝水
- 抬手擦汗
- 低头整理护具
- 脱下外套搭肩
- 冲刺后的停顿
- 训练后靠墙放空
- 衣服被汗轻贴身体时的自然状态

### 3. 年轻职业男性动作
优先动作：
- 办公桌前松领带
- 低头整理文件
- 会议结束后解开袖扣
- 端咖啡穿过办公区
- 快递员搬箱、扫码、交接签收
- 服务员俯身摆盘或整理围裙
- 安保扶对讲机巡逻
- 教师板书、翻书、停顿讲解
- 值班间隙独自坐着休息
- 下班后靠椅背放松肩颈

## 七、服装结构要求

服装不是标签，而是角色逻辑的一部分。

### 1. 校园 / 男大学生
优先：
- 白T
- 背心
- 球衣
- 宽松运动短裤
- 卫衣
- 轻薄衬衫
- 条纹衬衫
- 针织衫
- 学院风开衫
- 双肩包
- 运动袜
- 球鞋 / 跑鞋

### 2. 青春运动
优先：
- 篮球背心
- 足球训练服
- 跑步背心
- 紧身打底层
- 训练短裤
- 球队外套
- 护腕、发带、毛巾
- 汗湿后的上衣
- 训练后搭在肩上的外套

### 3. 职业 / 制服 / 工装
优先：
- 修身西装
- 白衬衫
- 马甲
- 领带
- 工牌
- 制服衬衫
- 安保制服
- 快递员工服
- 餐饮围裙
- 教师风针织和西裤
- 工装裤
- 制服外套
- 卷起的袖口
- 略微松开的领口

## 八、道具设计要求

道具必须参与叙事，不能只是摆设。

优先道具：
- 课本、笔记本、篮球、足球、水瓶、毛巾、耳机、背包、运动护具
- 文件夹、工牌、对讲机、围裙、托盘、纸箱、扫码枪、快递车、咖啡杯、教材、白板笔

检查标准：
- 拿掉道具后，人物是否还成立？
- 道具是否真的对应这个角色和场景？
- 道具是否帮助解释动作，而不是单纯凑元素？

## 九、性感表达的正确来源

BoyStyle 可以写吸引力、性感张力和轻幻想氛围，但必须来自以下成立的来源：
- 青春身体状态
- 运动后的呼吸感和汗意
- 制服或衬衫对肩背、腰线、手臂的包裹
- 卷袖口、松领口、整理衣摆等过渡瞬间
- 劳动中的力量感
- 服务关系中的克制感
- 权威身份中的温和压迫感
- 男大学生的自然、干净、未经雕琢的吸引力
- 运动员在动态与静止之间的身体张力

禁止把性感直接写成：
- 只有露肉
- 只有擦边
- 只有“嫩”“帅”“欲”这些抽象词
- 完全脱离身份逻辑的姿势化卖弄

## 十、必须避免的错误

### 1. 空词堆砌
禁止只写：
- 帅
- 嫩
- 男大
- 性感
- 制服诱惑
- 青春感

而不补充身份、场景、动作、服装和道具。

### 2. 场景悬空
禁止：
- 没有环境支撑
- 只有棚拍感
- 看不出为什么他会出现在这里

### 3. 职业假、制服假
禁止：
- 只是穿了类似制服的衣服
- 没有对应动作和环境
- 职业设定像 cosplay 而不是生活现实

### 4. 过度脱离站点气质
禁止：
- 过度成熟油腻
- 过度欧美化 bodybuilder 想象
- 过度重口制服 fetish
- 完全丢失校园、青春、运动、东亚青年语境

## 十一、推荐的 prompt 结构

后续凡是要写角色化 prompt、图像 brief、封面构想、内容策划说明，建议优先按这个结构组织：

1. 角色身份
2. 场景环境
3. 穿着结构
4. 动作设计
5. 道具细节
6. 身体状态
7. 氛围关键词
8. 吸引力来源

## 十二、最小发布检查清单

每次使用人物场景 prompt 前，至少检查：
1. 是否一眼能看出角色身份？
2. 是否一眼能看出场景？
3. 是否有成立的动作，而不是静态摆拍？
4. 服装是否与身份相符？
5. 是否保留青春 / 运动 / 生活流气质？
6. 吸引力是否来自具体语境，而非空词？
7. 是否仍然属于 BoyStyle，而不是跑偏到别的站点气质？

## 十三、当前执行策略

在后续扩展 prompt、做人物图、整理题图方向和规划内容专题时：
- 校园青春线与运动青春线应继续作为基础盘
- 职业 / 制服 / 工装内容可以扩展，但必须建立在“年轻男性真实场景化表达”上
- 男大学生、运动员、实习生、办公室职员、快递员、服务员、安保、教师可作为第一批重点角色库
- 后续如需批量生产，应把本规范继续拆成：角色库、场景库、动作库、服装库四套可组合模板
